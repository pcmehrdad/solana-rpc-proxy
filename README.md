
## Solana RCP proxy

This project is a dockerized version of [this repo](https://github.com/HowRareIs/solproxy)

## how to use
```bash
git clone this-repo
cd to it
cp .env.example .env
cp config.json.example config.js
docker compose build
docker compose up -d

```
